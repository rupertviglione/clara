# clara

Site institucional do estúdio **clara**.  
Presença digital focada em clareza, desempenho e credibilidade.

Este projecto evita frameworks e dependências desnecessárias.  
O objectivo é controlo total sobre código, performance e mensagem.

---

## O que é

Um site estático desenvolvido em **HTML, CSS e JavaScript puro**, pensado para:

- projectos criativos e culturais
- estúdios, freelancers e pequenas estruturas
- clientes que precisam de presença digital séria, não de pirotecnia

---

## Princípios

- **Clareza antes de estética**
- **Performance antes de efeitos**
- **Código simples, legível e durável**
- **Semântica e acessibilidade como base**
- **Manutenção fácil a longo prazo**

Nada aqui é experimental. Tudo é intencional.

---

## Stack técnica

- HTML5 semântico
- CSS único (`assets/styles.css`)
- JavaScript vanilla (`assets/app.js`)
- Sem frameworks
- Sem build tools
- Sem dependências externas
